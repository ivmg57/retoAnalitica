Cómo correr el código de análisis estadístico en Python

Preparativos:

Asegúrate de tener Python instalado en tu sistema. Si no lo tienes, puedes descargarlo desde python.org.
Instala la biblioteca pandas.
Asegúrate de tener el archivo peso_estatura_genero.csv en la misma carpeta donde planeas guardar y ejecutar el script de Python.

Leer el output:

Información básica del conjunto de datos: Se mostrará el número de entradas, el tipo de cada columna y el uso de memoria.
Resumen estadístico: Aparecerán varias estadísticas como la media, la mediana, la desviación estándar, y los valores mínimos y máximos para las variables "Estatura" y "Peso".
Métricas adicionales: Se mostrarán varias métricas como la media recortada, la media ponderada, la moda, la mediana, la varianza, la desviación estándar, y los cuartiles de la variable "Estatura".
